Greedy Algorithms: Scheduling Classes

  • Scheduling Classes problem
  • Recursive solution
  • O(n3n^3) time
def GreedySchedule(S, F):
  sort arrays by ascending end time
  count = 1
  X[count] = 1
  for i=2 to n:
    if S[i] > F[X[count]]:
      count += 1
      X[count] = i
  return X

Finds maximum number of non-overlapping classes.

Lemma: At least one maximal conflict-free schedule includes the class that finishes first.

Proof: Let ff be the class that finishes first. Suppose we have a maximal conflict-free schedule XX that does not include ff. Let gg be the the first class in XX to finish. Since ff finishes before gg, ff cannot conflict with any class in the set XX without gg. X with ff instead of gg is the same size, so it is also a maximal solution.

Theorem: The greedy schedule is an optimal schedule.

Proof: Let (g1,g2,gj−1,…,cj,cj+1,…,cm)(g_1, g_2, g_{j-1}, \ldots , c_j, c_{j+1}, \ldots, c_m) by a maximal conflict-free schedule, where classes gig_i are chosen by a greedy algorithm, and cic_i chosen otherwise.

By construction, the jjth greedy choice gjg_j does not conflict with earlier classes, and neither does cjc_j since our schedule is conflict-free.

Additionally, gjg_j has the earliest finish time among all classes that don’t conflict with earlier classes; it finishes before cjc_j. Thus, the following schedule is also conflict-free:

(g1,g2,gj−1,…,gj,cj+1,…,cm)(g_1, g_2, g_{j-1}, \ldots , g_j, c_{j+1}, \ldots, c_m)

We may repeat the same argument for all remaining classes that are not chosen by the greedy algorithm.
